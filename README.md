
# Word Game Client

## Usage

 - Clone this repo locally.
 - Edit play.js to set your name.
 - Edit play.js to make a good player for the game.
 - Make sure you've got node.js installed.
 - Install deps with ```npm install```
 - Run with ```node play.js```
